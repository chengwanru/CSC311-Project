MarkUs — Report assignment → code.zip
========================================

1. Run from repo root:  python submission/sync_submission.py

2. Create code.zip from the CONTENTS of this folder:
   - Select everything inside 02_report_code_zip/ (all .py files, appendix_code/, LICENSE, requirements-figures.txt, etc.)
   - Compress into code.zip
   - Do not include this README.txt if you prefer a minimal zip (optional).

3. Upload code.zip to the report slot on MarkUs.

Evidence only — TA does not need to run it. Include appendix_code/ if your report discusses alternative setups.
